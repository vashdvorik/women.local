# Изображения главной страницы

Изображения созданы встроенным `imagegen` в режиме `photorealistic-natural`. Исходный концепт заказчика использовался только как визуальный референс для hero. В generated-файлах нет текста, логотипов, интерфейса и водяных знаков.

## Набор файлов

| Файл | Использование | Финальный prompt |
|---|---|---|
| `hero-community.webp` | Hero | Четыре женщины-предпринимательницы из Восточной Европы совместно работают за ноутбуком в светлом современном офисе; широкий кадр 3:2, естественный дневной свет, нейтральная одежда, живое деловое взаимодействие |
| `member-fashion.webp` | Профиль: мода | Естественный деловой портрет предпринимательницы около 35 лет в бежевом жакете, светлый нейтральный офисный фон, тёплое уверенное выражение |
| `member-agrifood.webp` | Профиль: агробизнес | Естественный деловой портрет опытной предпринимательницы около 40 лет в белой блузке, мягкий свет и едва заметная зелень на фоне |
| `member-digital.webp` | Профиль: IT | Естественный деловой портрет предпринимательницы около 30 лет в тёмно-зелёном жакете, современный светлый офисный фон |
| `event-workshop.webp` | Бизнес-школа | Пять женщин обсуждают бизнес-планы с ведущей в светлом учебном помещении; широкий репортажный кадр 16:9 |
| `event-esg.webp` | ESG-вебинар | Ноутбук с абстрактной презентацией устойчивого развития, зелёные природные формы и рука участницы с блокнотом; широкий кадр 16:9 |
| `event-networking.webp` | Региональный форум | Разнообразная группа профессиональных женщин общается в networking-зоне современного форума; естественные жесты, широкий кадр 16:9 |
| `story-mentor.webp` | История наставничества | Предпринимательница-наставница в тёмно-синем жакете в современном рабочем кабинете; горизонтальный editorial portrait 4:3 |
| `story-export.webp` | История экспорта | Предпринимательница локального бренда в бежевом жакете с приглушённым бирюзовым акцентом в светлой студии; горизонтальный editorial portrait 4:3 |

Общие ограничения prompt set: реалистичная кожа и материалы, естественные лица и руки, отсутствие позированной stock-стилистики, текста, брендов, логотипов, watermark и чрезмерной ретуши.

## Оптимизация

- формат: WebP;
- качество: 84;
- общий вес девяти изображений: около 539 КБ;
- hero загружается с `fetchpriority="high"`;
- остальные изображения используют `loading="lazy"` и `decoding="async"`;
- HTML содержит реальные размеры файлов для предотвращения layout shift.

## Ограничение использования

Сгенерированные портреты являются демонстрационными. Перед публичным запуском профили и истории должны использовать фотографии реальных участниц только после получения согласия и подтверждения прав на публикацию.
